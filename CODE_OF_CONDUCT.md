# Code of Conduct

## Our Pledge
We are committed to making this project welcoming and respectful for everyone.

## Expected Behavior
- Be respectful and constructive.
- Focus feedback on code and ideas, not people.
- Help others learn and contribute.

## Unacceptable Behavior
- Harassment, discrimination, or abusive language
- Doxxing, threats, or other harmful behavior
- Disruptive or intentionally hostile conduct

## Scope
This code of conduct applies to issues, pull requests, comments, and any project-related communication.

## Enforcement
Project maintainers may remove or edit content and may restrict participation for behavior that violates this code of conduct.
